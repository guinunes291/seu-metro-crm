## Mecânicas do Sistema

**Visual das Conquistas:**
- **Bloqueadas**: Ícone cinza, transparência 30%, barra de progresso vazia
- **Em Progresso**: Ícone colorido, transparência 60%, barra de progresso parcial
- **Completas**: Ícone colorido brilhante, sem transparência, badge dourado

**Notificações:**
- Ao completar uma conquista, o corretor recebe notificação push
- Toast aparece na tela com animação de confete
- Som de celebração é reproduzido

**Celebração:**
- Animação de confete colorido por 3 segundos
- Som de "tada" ou fanfarra
- Modal de parabéns com detalhes da conquista
